检测栈

Source / Implementation

先抓已验证源码；代理合约会继续跟踪实现合约，避免源码和字节码分析目标错位。

Slither

源码存在时执行静态检测，优先覆盖常见 Solidity 缺陷与代码味道。

Mythril

通过 RPC 直接分析链上字节码，没有源码时仍可给出基础漏洞信号。

使用边界

这是一个第三方分析器聚合台，不会因为“未报问题”就等于安全。目标是更可靠地发现基础漏洞，不替代正式审计。
